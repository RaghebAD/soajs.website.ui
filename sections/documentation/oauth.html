SOAJS OAUTH


SOAJS oAuth is a service that integrates with oAuth 2.0.
The service simply validates, refreshes and kills access tokens generated via oAuth 2.0. This service is useless by itself and is used by other services that require oAuth 2.0 security. When developing a SOAJS service, a configuration is provided to SOAJS, this configuration should contain < "oauth":true >, then this service cannot be accessed unless an oAuth 2.0 access token is provided in every request made to this service.

Installation

SOAJS oAuth is located in soajs.oauth.
Install:

$ npm install soajs.oauth

Then navigate to the directory where it is installed and run the service.
$ cd soajs.oauth
$ node index

The service is now running and accessible vial the controller on http://localhost:4000/oauth.
