SOAJS Agent

Will be available soon.

SOAJS Agent is a service that empowers SOAJS with push mechanisms, maintenance operations and analytics integration.
The Agent can update the topology of the application on different environments.
The Agent can perform maintenance operations on all services, and reload their configuration.
The Agent is also responsible for sending data to the analytics systems and maintaining SOAJS's awareness.

Features

  1. Topology Updates:
    Push SOAJS service topology including installed services, packages, dependencies and versions to the dashboard.

  2. Maintenance:
    Trigger maintenance operations such as heartbeat checks and provision reloading for all SOAJS services.

  3. Service Analytics:
    Capture and send all SOAJS services logs to elastic search for analytics updates.

  4. System Analytics:
    Capture and send SOAJS controller logs to elastic search.

  5. System Information:
    Retrieve and provide Operating System, Input/Output and Network Analytics information to dashboard.

  6. Awareness:
    Keep self awareness among services up-to-date.

Installation

SOAJS Agent is located in soajs.util.
Install:

$ npm install soajs.agent

Then navigate to the directory where it is installed and run the service.
$ cd soajs.agent
$ node index

The service is now running and accessible via the controller on http://127.0.0.1:4000/agent/.
