SOAJS Agent

SOAJS Agent is a service that empowers SOAJS with push mechanisms and analytics integration.
The Agent can update the topology of the application on different environments.
The Agent is also responsible for sending data to the analytics systems and maintaining soajs's awareness.

Features:

  1. Topology Updates:
    Push soajs service topology including installed services, packages and versions to the dashboard.

  2. Maintenance:
    Trigger maintenance routes for all local soajs services.

  3. Service Analytics:
    Capture and send all local soajs services logs to elastic search for analytics updates.

  4. System Analytics:
    Capture and send soajs controller logs to elastic search.

  5. System Information:
    Retrieve and provide Operating System, Input/Output and Network Analytics information to dashboard.

  6. Awareness:
    Keep self awareness among services up-to-date.

Usage:

SOAJS Agent is located in soajs.agent.
Install:

$ npm install -d soajs.agent

Then navigate to the directory where it is installed and run the service.
$ cd soajs.agent
$ node index

The service is now running and listening http://127.0.0.1:4000/agent/.
