SOAJS OAUTH

Installation

SOAJS oAuth is located in soajs.oauth. 

$ cd /opt/soajs/node_modules/
$ npm install soajs.oauth
$ cd soajs.oauth
$ node .

The service will listen on port 4002 to handle requests and 5002 for maintenance operations and is accessible via the controller on http://127.0.0.1:4000/oauth.

SOAJS oAuth is a service that integrates with oAuth 2.0. The service simply validates, refreshes and kills access tokens generated via oAuth 2.0. Services that contain "oauth": true when declared, will be protected by oAuth 2.0 and require an access_token that is generated by this service to be available in all requests that they receive so that these requests gets processed. Any request missing an access_token or providing an invalid one will be blocked.
