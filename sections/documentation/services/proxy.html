SOAJS PROXY

Installation

SOAJS proxy is located in soajs.prx. 

$ cd /opt/soajs/node_modules/
$ npm install soajs.prx
$ cd soajs.prx
$ node .

The service will listen on port 4009 to handle requests and 5009 for maintenance operations and is accessible via the controller on http://127.0.0.1:4000/proxy.

SOAJS proxy is a service that redirects requests to the requested environments. These requests are made from modules developed under the Operate pillar to your services that are deployed in the requested environments. This enables the dashboard to provide you with the ability to use the Operate pillar via the frontend UI SDK and make calls to your custom built services in other environment.
