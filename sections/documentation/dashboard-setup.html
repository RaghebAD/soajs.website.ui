Dashboard Setup

Installation:

To install the dashboard, install all the needed modules of SOAJS and their depenencies as mentioned in Get Started Section under How to Install SOAJS.

Once all the modules are installed and the services are running, make sure that you have nginx installed and configured as mentioned in soajs.nginx so that the domains and hosts needed are accessible for you to start working.

Import Test Data:

Install soajs.examples.
Navigate to soajs.example/tools in your Terminal, and you will find a mongo script that imports some test data for you to start with.

# make sure soajs.mongo.sh is executable
$ chmod +x soajs.mongo.sh

# execute soajs.mongo.sh
$ ./soajs.mongo.sh

The script will import some test data so that you can login to the dashboard and start using it.
Note: You can always use the dashboard without importing the test data.


Using the Dashboard UI:

Since nginx has been configured as mentioned in this Link, you can now open your browser and use the Dashboard UI by going to: http://dashboard.soajs.org.

To Login, you need to provide a tenant key like the one obtained after you imported the test data, please checkout example03 in the Get Started section for additional help on how to use a key with a service.
The tenant key should be provided in the header of every request made to the dashboard service.

You can always lookup the tenant information under Provisioning section.

Once logged in, you can access all the sections and featured mentioned in the Dashboard Services Section and manage them accordingly.


Dashboard UI Features:

The Dashboard interface is built on AngularJS.

Benefit from the smart tools the dashboard offers to easily navigate and manage SOAJS features.

Menus and Grids enable you navigate, search, sort and browse the different sections and features.
Quick links to Add, Edit and Remove content and/or configuration.

Interactive forms and notification messages offer you all schemas needed to modify the configuration and invoke related API services.
