SOAJS News

River Shark Release contains the latest SOAJS updates and the first release in its new 1.x version.
In this new version, SOAJS introduced the new sessionless behavior where logged in users no longer have sessions unless required by the microservices to store custom product information.
In addition, soajs framework is broken down to multiple repositories for better management, flexibility, modulatiry and usage.
Please use the latest version of the SOAJS Installer to deploy this new version.

The new SOAJS version is not backward compatible.
