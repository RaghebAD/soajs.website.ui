SOAJS in the news

Release 2
20 April 2015
Release Notes:
  • soajs version 0.0.2
    SOAJS framework.
    Bug Fixes:
    • Updated how headers content-type are read in imfv and parsed.
    Enhanced Tests:
    • Added soajs.mongodb.data as dev dependency and integrated it in test cases.
    • Removed old provisioned data from module.
  • soajs.utilities version 0.0.2
    SOAJS utilities repository.
    Features:
    • Created deployment AIO configuration data files.
    • Created deployment cloud & awareness configuration data files.
  • soajs.urac version 0.0.2
    SOAJS User Registration and Access Control.
    Updates:
    • Added support to manage members groups.
    • Updated login API and how dashboard ui permissions are being fetched.
    • Added locked feature for user and group records to prevent deletion of administrator account from dashboard ui.
    • Updated IMFV configuration to include new format for username parameters.
    Enhanced Tests:
    • Added new tests for group APIs
    • Updated email values in test cases.
    • Added soajs.mongodb.data as dev dependency and integrated in test cases.
    • Removed old provisioned data from module.
  • soajs.dashboard version 0.0.2
    SOAJS dashboard service and User Interface.
    Updates:
    • Added locked feature for environment, product and tenant. Locked records cannot be deleted.
    • Updated how permissions are read after login in the UI.
    • Updated UI form library.
    • Under Productization section, combined the management of products and packages under one section.
    Enhanced Tests:
    • Updated test data used in test cases.
    • Added soajs.mongodb.data as dev dependency and included it in test cases.
    • Removed old provisioned data from module.
    Features:
    • Added new APIs to add/manage oauth users in service.
    • Added UI support for Urac group management APIs.
  • soajs.oauth version 0.0.3
    SOAJS oAuth Service.
    Enhanced Tests:
    • Added soajs.mongodb.data to dev dependency and included it in test cases.
    • Removed old provisioned data from module.
  • soajs.examples version 0.0.2
    SOAJS Services Examples Repository.
    Enhanced Tests:
    • Added test cases to increase coverage.
    • Added soajs.mongodb.data as dev dependency and included it in test cases.
    • Removed old provisioned data from module.
  • soajs.mongodb.data version 0.0.1
    SOAJS Combined Test Data Repository.
    Features:
    • Created test data content for each module.
    • Updated mail service configuration to use sendmail.

Release 1
20 February 2015
Release Notes:
  • soajs version 0.0.1
    SOAJS framework.
    Features:
    • Multi services with cloud self awareness
    • Multi projects, environments and configuration
    • Input mapping formatting & validation (IMFV)
    • Services APIs Security via tenant keys and oAuth 2.0
    • Multitenancy & Productization
  • soajs.controller version 0.0.1
    SOAJS main gateway service.
    Features:
    • Forwards requests to SOAJS services
  • soajs.utilities version 0.0.1
    SOAJS utilities repository.
    Features:
    • Contains Nginx configuration instructions.
  • soajs.urac version 0.0.1
    SOAJS User Registration and Access Control.
    Features:
    • Registration & Login for Guest users
    • Profile Management of logged in members.
    • Members Accounts Management for Administrators.
  • soajs.dashboard version 0.0.1
    SOAJS dashboard service and User Interface.
    Features:
    • Management of products, packages and access levels in Productization.
    • Management of tenants, applications, keys and service configuration in Multitenancy.
    • Creation of multiple Environments.
  • soajs.oauth version 0.0.2
    SOAJS oAuth Service
    Features:
    • Services and their APIs protection using oAuth 2.0
  • soajs.examples version 0.0.1
    SOAJS Services Examples Repository.
    Features:
    • Provides examples on how to create basic services.
    • Provides examples on how to create multitenant services.
    • Provides examples on how to create protect services with oAuth 2.0.
  • connect-mongo-soajs version 0.0.1
    SOAJS stand alone Connect Mongo Driver.
    Features:
    • Provides full control over Mongo configuration including replica set support.
