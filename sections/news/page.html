Release 1
20 February 2015
Release Notes:
  • soajs version 0.0.1
    This is the first major release of soajs framework.
    Features:
    • Support for multiple services with cloud self awareness
    • Support for multiple projects, environments and configuration
    • Input mapping formatting & validation (IMFV)
    • Ability to secure services
    • Support for multitenancy & productization
  • soajs.controller version 0.0.1
    This is the first release of the soajs gateway controller.
    Features:
    • Forwards requests to SOAJS services
  • soajs.utilities version 0.0.1
    This is the first release of the soajs.utilities repo. It contains nginx configuration instructions.
  • soajs.urac version 0.0.1
    This is the first release of the soajs urac service.
    Features:
    • Guest users can join and login
    • Members can manage their accounts
    • Administrators can manage users
  • soajs.dashboard version 0.0.1
    This is the first release of the soajs.dashboard service. The dashboard service allows you to manage SOAJS components.
    Features:
    • Manage products in Productization section
    • Manage tenants and keys in Multitenancy section
    • Create multiple environments
    • Create tenant applications, keys and external service API keys
  • soajs.oauth version 0.0.2
    This is the first release of the soajs.oauth service. It is a service used to protect other services using oAuth 2.0.
  • soajs.examples version 0.0.1
    This is the first release of soajs.examples. It contains a list of services that were created to demonstrate the use of SOAJS framework.
  • connect-mongo-soajs version 0.0.1
    This is the first release of connect-mongo-soajs. It is a stand alone SOAJS mongo driver.
    Features:
    • Gives you full control over mongo configuration including replica set support
